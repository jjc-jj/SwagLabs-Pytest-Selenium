# 环境部署与网络替补方案

> 适用场景：十堰网络环境，访问境外服务器（Google、Apache、Postman 等）可能不稳定
> 原则：**每个工具至少准备 3 条下载路径，从不依赖"碰运气"**

---

### 1. Python 库（selenium, pytest, pytest-html, requests, pytest-rerunfailures, responses, webdriver-manager）

**一次性全部安装命令**：

```bash
pip install selenium pytest pytest-html requests pytest-rerunfailures responses webdriver-manager -i https://pypi.tuna.tsinghua.edu.cn/simple
```

| 方案 | 命令/地址 | 适用条件 |
|------|----------|---------|
| **主方案：清华镜像** | `-i https://pypi.tuna.tsinghua.edu.cn/simple` | 国内网络正常即可 |
| 替补 A：阿里云镜像 | `-i https://mirrors.aliyun.com/pypi/simple/` | 清华镜像挂了或慢 |
| 替补 B：中科大镜像 | `-i https://pypi.mirrors.ustc.edu.cn/simple/` | 前两个都不可用 |
| 替补 C：离线 .whl 包 | 从 https://pypi.org/ 搜包名 → Download files → 下载 .whl → `pip install xxx.whl` | 全部镜像不可达 |
| **终极兜底** | 用手机热点 + pip 清华镜像（手机 4G/5G 通常不受校园网限制） | 校园网/宽带全面封锁 |

**验证安装成功**：
```bash
python -c "import selenium; import pytest; print('ALL OK')"
```

### 5. ChromeDriver

> ⚠️ 这是**整个环境搭建中最容易失败的一步**。Google 的 storage.googleapis.com 在部分地区完全无法访问。

**第零步：确认 Chrome 版本**
```
Chrome 地址栏输入：chrome://version
第一行显示：Google Chrome: 126.0.6478.127 (正式版本) ...
记下主版本号：126
ChromeDriver 必须下载 126.x.x.x 的版本！
```

| 方案 | 地址 | 说明 |
|------|------|------|
| **主方案：淘宝 npm 镜像** | https://registry.npmmirror.com/binary.html?path=chromedriver/ → 选对应版本号 → `chromedriver_win32.zip` | 国内 CDN，十堰稳定 |
| 替补 A：webdriver-manager（自动匹配） | `pip install webdriver-manager -i https://pypi.tuna.tsinghua.edu.cn/simple` → 代码中自动下载 | pip 能通就能用，首次仍可能失败（也访问 Google） |
| 替补 B：**切 Edge + EdgeDriver** | https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/ | 微软托管，国内下载无障碍。Selenium 完全兼容 Edge |
| 替补 C：U 盘拷贝 | 让朋友从 ChromeDriver 官网或淘宝镜像下载后传你 | 最可靠 |
| **终极兜底** | **直接用 Edge + EdgeDriver**。Edge 是 Chromium 内核，和 Chrome 在 Selenium 中表现完全一致，代码只需改一行 `webdriver.Chrome()` → `webdriver.Edge()` | 零风险 |

**Edge + EdgeDriver 方案（推荐作为主方案！）**：
```python
# 无需担忧 ChromeDriver 下载问题，Edge 开箱即用
from selenium import webdriver
from selenium.webdriver.edge.service import Service as EdgeService
from webdriver_manager.microsoft import EdgeChromiumDriverManager

driver = webdriver.Edge(service=EdgeService(EdgeChromiumDriverManager().install()))
driver.get("https://www.saucedemo.com/")
```

> **建议**：考虑到十堰网络，直接把 **Edge + EdgeDriver 设为主方案**，ChromeDriver 降级为替补。面试时说"我用 Selenium 驱动 Edge 浏览器做自动化"完全不影响评价。

### 10. Django 本地测试环境

| 方案 | 命令 | 说明 |
|------|------|------|
| **主方案：pip 清华镜像** | `pip install django -i https://pypi.tuna.tsinghua.edu.cn/simple` | Django ~8MB，清华镜像秒下 |
| 替补 A：pip 离线包 | 下载 .whl → `pip install Django-5.x-py3-none-any.whl` | 镜像都挂时用 |
| 替补 B：conda 清华镜像 | `conda install django -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/` | 如果你有 conda |
| 替补 C：手机热点 + pip | 切手机 4G/5G 热点，`pip install django` | 运营商网络通常 DNS 不同 |
| **终极兜底** | Django 不需要联网开发。只要 pip 装上了，后续 `python manage.py runserver` 纯本地运行，零网络依赖 | 一次性安装 |

**快速搭建 Django Polls 命令**（装完 Django 后执行）：
```bash
django-admin startproject mysite
cd mysite
python manage.py startapp polls
# 编辑 polls/views.py, urls.py 后
python manage.py migrate
python manage.py runserver
# 访问 http://127.0.0.1:8000/
```

### 11. Docker（可选，若依部署用）

| 方案 | 地址/命令 | 说明 |
|------|----------|------|
| **主方案：Docker Desktop 清华镜像** | https://mirrors.tuna.tsinghua.edu.cn/docker-ce/win/stable/ | 校园网快 |
| 替补 A：阿里云 Docker 镜像 | https://mirrors.aliyun.com/docker-ce/win/stable/ | 阿里 CDN |
| 替补 B：WSL2 + Docker | 先启用 WSL2 → Microsoft Store 装 Ubuntu → `apt install docker.io` | 绕开 Docker Desktop |
| 替补 C：不装 Docker，直接本地跑 | 若依需要 Java + MySQL + Redis。直接装这三个本地环境即可，不用 Docker | 反而更简单 |
| **终极兜底** | **不部署若依，只用 Django Polls**。Django Polls 自带 SQLite，一行 `python manage.py migrate` 数据库就绪，零额外依赖。数据库断言在 SQLite 上同样有效 | 推荐作为主方案 |

> **强烈建议**：**放弃 Docker，放弃若依**。Django Polls 已经能完美满足方案中所有需要本地数据库的场景（数据库断言、自动化回归）。Docker + 若依的部署复杂度远高于 Django，且对项目含金量没有增量帮助。
