⚠️ 重要说明（面试时主动提及）：
Swag Labs 为纯前端 Demo 站点，无真实后端 API。登录认证、购物车管理、订单处理均为客户端 JavaScript 实现。
这意味着：
• "API 接口"实际上是页面加载 + 前端逻辑，而非独立的 RESTful 端点
• Session Token 存储在 Cookie 中（名为 session-username），用于页面访问控制
• 越权测试的核心：不登录直接访问 /inventory.html、/cart.html、/checkout-step-one.html
面试话术："我在测试前先通过 DevTools 摸清了整个技术实现——发现这是一个纯前端应用，认证靠 Cookie、购物车靠本地状态。这反而让我的安全测试更有针对性，比如验证了未登录状态下直接 URL 访问是否会被拦截。"
